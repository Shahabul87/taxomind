# PRISM Student Analytics

## What It Does
Analyzes individual student learning performance using the PRISM framework.
Computes a Bloom's cognitive map, identifies fragile knowledge, classifies
the student's cognitive cluster, and generates actionable prescriptions.

## When to Use
- User asks "How am I doing?" or "Show my analytics"
- User wants to see their learning progress or performance
- User asks about their Bloom's level, cognitive profile, or growth
- User wants study recommendations or learning suggestions
- User mentions fragile knowledge, weak areas, or struggles
- User asks "What should I study next?"

## Capabilities
- Bloom's Cognitive Map (ceiling, growth edge, velocity)
- Cognitive Cluster classification (5 types)
- Fragile Knowledge detection (correct but shallow reasoning)
- AI-powered interpretive analysis
- Priority-ordered prescriptions with ARROW phases
- Student-friendly PRISM report generation
- Quick Snapshot mode (zero AI calls, instant)

## Required Information
1. Analysis depth (quick_snapshot / standard / deep_analysis)
2. Course scope (all_courses / specific_course / recent_activity)
3. Time range (last_7_days / last_30_days / last_90_days / all_time)

## Output
- Bloom's Cognitive Map with per-level mastery
- Cognitive cluster classification with explanation
- Alerts for fragile knowledge, declining mastery, study gaps
- Priority prescriptions with effort/impact estimates
- Student-friendly report with verification questions
- SAM Goal and ExecutionPlan tracking
