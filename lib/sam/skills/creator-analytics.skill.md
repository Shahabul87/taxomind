# PRISM Creator Analytics

## What It Does
Analyzes course-level and cohort-level performance using the PRISM framework.
Computes Bloom&apos;s distribution across cohort, identifies content quality issues,
root causes of student outcomes, and generates ROI-scored prescriptions for
course improvement.

## When to Use
- Creator asks "How are my students doing?" or "Course analytics"
- Creator wants to see cohort cognitive health or engagement
- Creator asks about dropout risk, retention, or completion rates
- Creator wants to analyze content or assessment effectiveness
- Creator asks for course improvement suggestions
- Creator mentions student struggles, at-risk students, or failing cohort
- Creator wants a course health report

## Capabilities
- Cohort Bloom&apos;s distribution and bimodal detection
- Fragile knowledge alarm across cohort
- Content and assessment quality analysis
- Root cause analysis (CONTENT/PEDAGOGY/ASSESSMENT/STUDENT/SYSTEM)
- ROI-scored prescriptions with ARROW phase references
- Dropout risk predictions with intervention windows
- Creator-friendly PRISM report generation

## Required Information
1. Course to analyze (specific course or all courses)
2. Time range (last_7_days / last_30_days / last_90_days / all_time)
3. Focus area (cognitive_health / engagement / content_quality / predictions / comprehensive)
4. Analysis depth (overview / standard / deep_dive)

## Output
- Cohort cognitive analysis with health score
- Content and assessment quality report
- Root causes with causal chains
- Priority prescriptions with ROI estimates
- Creator-friendly report with key metrics and next steps
- SAM Goal and ExecutionPlan tracking
