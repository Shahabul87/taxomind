
### Thumbnail Style Guide
- **Font**: Bold, condensed sans-serif (Impact style)
- **Text**: Maximum 5-6 words
- **Colors**: High contrast (white/yellow text on dark)
- **Face**: Include human face when possible (increases CTR)
- **Branding**: Consistent corner logo placement
- **Emoji**: Use sparingly for emphasis

### Thumbnail Templates by Content Type

**Tutorial Videos**
- Blue gradient background
- "HOW TO" text badge
- Screenshot or icon of feature
- White bold text

**AI Features**
- Purple/blue neural network background
- Robot/AI icon
- "SAM AI" text badge
- Glowing effect elements

**Tips/Quick Videos**
- Orange/yellow accent
- Number badge (e.g., "5 TIPS")
- Checkmark icons
- Bright, energetic feel

**Updates**
- Green accent
- "NEW" badge
- Screenshot of feature
- Arrow pointing to feature

## 17. BRAND VOICE GUIDELINES

### Tone
- **Professional** but approachable
- **Enthusiastic** about learning and technology
- **Supportive** and encouraging
- **Clear** and jargon-free when possible
- **Inclusive** - education is for everyone

### Language Style
```
✅ DO:
- "Let's explore how SAM helps you learn..."
- "You'll discover that adaptive learning..."
- "Here's a quick tip to boost your progress..."
- "We're excited to show you..."

❌ DON'T:
- "This is super easy, anyone can do it..."
- "Obviously, you already know..."
- "Trust me, this is the best..."
- "Other platforms can't compete..."
```

### Key Phrases to Use
- "Learn smarter, not harder"
- "Your personalized learning journey"
- "AI-powered education"
