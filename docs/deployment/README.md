# Deployment Documentation

This directory contains deployment guides for Taxomind.

## Available Deployment Guides

### Railway Deployment
- `RAILWAY_BUILD_QUICKSTART.md` - Quick start guide for Railway deployment
- `LOCAL_RAILWAY_BUILD_GUIDE.md` - Local Railway build testing

### Build Fixes
- `BUILD_FIX_SUMMARY.md` - Summary of build fixes and optimizations

## Deployment Workflow

1. **Local Testing**: Test build locally first
2. **Environment Setup**: Configure environment variables
3. **Build Verification**: Run production build
4. **Deploy**: Deploy to Railway or other platform
5. **Monitor**: Check deployment logs and health
